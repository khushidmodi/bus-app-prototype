# CampusFlow Prototype

Static frontend prototype for a campus bus tracking application with a dark, PassioGo-inspired UI.

## Run

Open [index.html](/Users/khushimodi/bus-app-prototype/index.html) directly in a browser, or serve the folder with a simple static server.

## Files

- [index.html](/Users/khushimodi/bus-app-prototype/index.html): app shell
- [src/app.js](/Users/khushimodi/bus-app-prototype/src/app.js): state, rendering, interactions, simulation
- [src/data.js](/Users/khushimodi/bus-app-prototype/src/data.js): mock routes, stops, buildings, timetables
- [src/styles.css](/Users/khushimodi/bus-app-prototype/src/styles.css): dark UI, animations, map and sheet styling
